# Medical-Device

## In this project we are going to learn how to code and use arduinos by creating a work medical device that can read your heart rate and check your temperature.

# Parts List:
* Arduino UNO
* LM35 sensor
* jumper wires
* small bread board
* pulse sensor
* LCD Screen 16x2 with I2C
